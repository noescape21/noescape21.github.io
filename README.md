# Muhammad Saad — Portfolio (Static)

## Files
- index.html      : Single-file React portfolio (CDN React + Tailwind + marked.js)
- robots.txt      : Allows all crawlers, advertises sitemap
- sitemap.xml     : Sitemap (update <loc> with absolute URL once domain is set)
- llms.txt        : Summary for LLM crawlers
- posts/          : Drop your blog markdown files here (e.g. attack-paths.md)

## Deploy
Upload all files to any static host (GitHub Pages, Netlify, Cloudflare Pages, S3).
After publishing, replace the relative URLs (`/`) in sitemap.xml and the
canonical/og:url tags in index.html with your absolute domain (e.g. https://saad.dev/).
